# Maze
A game where you have to escape 3 consecutive mazes to get to the pie. Each maze is different and has different unique challenges.

## Includes
*   **Easy to learn** Clear instructions provided at the start with only the arrow and space keys being used
*   **Fun visuals** A bright and colourful game with a rainbow trail of balls following you
*   **Changable game difficulty** Play the mazes with either fog or teleporters
*   **Trackable High Scores** The times for each maze for each difficulty are tracked leading to having the highscores shown at the end of the game

## How to play

*   Read the instructions at the start of the program
*   Choose which difficulty the game is (fog, teleporters, neither)
*   Try to escape the mazes by moving through using the arrow keys
*   If the are teleporters, teleport by touching a teleporter and clicking space
*   Game ends when you have sucessfully escaped all 3 mazes
*   The games are replayable by clicking on the buttons with the times per game.
